SimpleTree

By Jacob Marshall

SimpleTree is an extremely efficient plugin. It takes jQuery and mixed some incredible logical CSS, and packs a huge punch. SimpleTree was originally developed for educational purposes only, but I though that I may as well release it, because it would probably end up going to waste otherwise.
